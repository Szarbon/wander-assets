## Design notes
- Typeface: UnifrakturCook-Bold (Old English blackletter)
- Mark: capital W with a horizontal currency stripe across the middle (currency glyph tradition — like `$` has a vertical stripe)
- Palette: pure black + white (no gradients, no colors)
- Format: 512×512 master, rounded-square with 19% corner radius for token-list compatibility
